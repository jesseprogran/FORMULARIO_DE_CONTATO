## Sobre o Projeto

O projeto "Formulário de Contatos" foi desenvolvido com o objetivo de criar uma experiência positiva para os usuários ao enviar suas informações de contato. Utilizando HTML, CSS e JavaScript, implementamos um formulário interativo com as validações necessárias para garantir a integridade dos dados coletados.

Esperamos que este projeto seja útil e facilite a comunicação entre os usuários e a equipe responsável pelo recebimento de contatos. Agradecemos a todos pela atenção e estamos disponíveis para esclarecer quaisquer dúvidas que possam surgir. Obrigado!
### Tecnologias Utilizadas

- HTML
- CSS
- JavaScript


### Funcionalidades

- Campos de entrada: Nome, e-mail, assunto 
e mensagem.
- Validações: Verificação de campos obrigatórios e formato de e-mail.
- Limitação de caracteres na mensagem.
- Confirmação de envio antes do envio definitivo.

## Como Utilizar

### Pré-requisitos

- Navegador Web moderno (Chrome, Firefox, Safari, etc.)

### Instalação

1. Clone o repositório em seu computador: https://github.com/jesseprogran/FORMULARIO_DE_CONTATO

2. Abra o arquivo `index.html` em seu navegador web.
